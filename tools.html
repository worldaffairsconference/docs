Text Editor

A text editor is a developer's best friend, and you'll definitely need one to do any development work somewhat efficiently. Here's a set of text editors that we recommend (no specific one is mandatory, though Matt prefers Atom).

Atom

Advantages: many packages and themes, large community, good git integration

Atom Website and Download

Sublime Text

Advantages: many packages and themes, large community, easy customization

Sublime Text Website and Download

Brackets

Advantages: many extensions, good web design integration, customized adobe plugins

Brackets Website and Download

Dependencies and Libraries

Since you'll probably need to do both frontend and backend work anyways, we recommend you install both of their dependencies and libraries. You'll also notice that they share a lot of them, and that's for a reason.

Software
